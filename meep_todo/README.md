# meep_todo

### How to run?
1. flutter pub get
2. flutter pub run build_runner build  
3. flutter run

### How to test?
1. flutter test
2. flutter drive \
  --driver=test_driver/perf_driver.dart \
  --target=integration_test/app_test.dart \
  --profile --no-dds

### features
1. local storage
2. widget test
3. profile intergation test
3. android CICD
